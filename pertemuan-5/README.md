### Spesifikasi Teknis Pengembangan
1. **Framework CSS:** Menggunakan Bootstrap 5 via CDN.
2. **Backend:** Dibangun menggunakan Node.js dengan framework Express.
3. **Struktur Halaman:** Terdiri dari 3 halaman utama:
   - **Home (`/`)**: Halaman utama / dashboard.
   - **Form (`/form`)**: Halaman untuk input data (Create) dan edit data (Update).
   - **Data Tabel (`/data`)**: Halaman yang menampilkan seluruh data.
4. **Library Tambahan:** Menggunakan jQuery dan plugin jQuery DataTables.
5. **Manajemen Data:** Mengambil data berformat JSON dari endpoint API `/api/mahasiswa` dan dirender langsung ke dalam DataTable.
